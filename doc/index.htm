The ZicroniX Operating System

What is Zicronix?

Zicronix is a hobby operating system under development, it's main aim is to well...., teach the developer about
Operating System Development. It's original name was Bendex, but then it got changed to ZicroniX.
Zicronix is not all *NIX compatible, it's a completely new system.
